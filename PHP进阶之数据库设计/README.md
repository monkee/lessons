PHP进阶之数据库设计
=======

## 介绍

给你一个项目，如何做好设计？  
选择了DB，那么如何做表的设计？  
选择什么样的表引擎？索引如何建？  
表的字段如何加，考虑的因素有哪些？  
这门课主要讲解这些内容，学完这门课，你讲会做PHP在项目开发中，DB部分的设计方案。

## 课程内容

**第一节** 选择和使用MYSQL  
**第二节** 选择合适的表引擎  
**第三节** 设计索引  
**第四节** 字符集的处理  
**第五节** 常见的表结构（模型与关系）  
**第六节** 常见的缓存字段（空间换时间）  
**第七节** 使用事务  
**第八节** 分表
